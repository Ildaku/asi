# Planner2

## Как запустить

1. Активируйте виртуальное окружение:
   
   ```
   venv\Scripts\activate
   ```

2. Установите зависимости:
   
   ```
   pip install -r requirements.txt
   ```

3. Запустите приложение:
   
   ```
   python run.py
   ```

4. Откройте в браузере [http://127.0.0.1:5000](http://127.0.0.1:5000)
